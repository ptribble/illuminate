Tribblix Package Viewer

The Package Viewer allows you to study the SVR4 packages installed on your system.

There are 3 available views:

  • The Package List tab shows the packages installed, initially in alphabetical order of package name, although you can change the sort order using the column headers.
  • The Overlays tab shows the available overlays, their constituent packages, and whether they're installed.
  • The Filesystem tab allows you to browse the filesystem and see which packages and overlays contain a file.
The above views give general information on the package, the other packages it depends on and which depend on it, and which overlays the package is contained in.
