JFSstat

JFSstat is a tabular display of the filesystem statistics kept by illumos. It uses the same kstats as the fsstat command line utility.

The available kstats are extensive, and only a subset can be displayed at any time. Use the "Display" menu to choose which set of statistics to display.

The rows can be sorted by selecting the column header.

The toolbar

The toolbar contains a small number of menus.

  • File - allows you to exit jfsstat.
  • Update every - allows you to change how rapidly the statistics are updated.
  • Help - shows this help file, and the CDDL license that this software is licensed under.
  • Display - allows you to select which statistics (in other words, which filesystem operations) are shown.

  • Show - allows you to toggle the display of hidden filesystems and aggregates by filesystem type. By default, hidden filesystems and aggregates aren't shown.
