illuminate

illuminate is a graphical utility to display useful information about a computer running an illumos distribution.

The information shown includes:

  • Services
  • General information about the system
  • A system explorer view
  • Installed software

Toolbar

The toolbar contains a small number of menus.

  • File - allows you to exit illuminate, or launch a second copy.
  • Tools - allows you to launch external utilities that show information about your system.
  • Kstat Demos - launch a few useful tools showcasing JKstat
  • Help - shows this help file.
